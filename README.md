# Toy Store Capstone Project

A Laravel + React application for a toy store marketplace with admin, seller, and buyer roles.

## Introduction

This application provides a robust, modern starting point for building a toy store marketplace with a React frontend using [Inertia](https://inertiajs.com).

Built with Laravel backend and React 19 frontend using TypeScript, Tailwind CSS, and modern UI components.

## Features

- **Multi-role Authentication**: Admin, Seller, and Buyer roles
- **Dashboard System**: Role-based dashboards for different user types
- **Modern UI**: Built with shadcn/ui and Radix UI components
- **Two-Factor Authentication**: Enhanced security features
- **Responsive Design**: Mobile-friendly interface
